## 1. Цель продукта и общая концепция

**Название:** ClayShield  
**Тип:** iOS Utility App  
**Цель:** создать эстетичное, простое и эффективное приложение для защиты пользователя от рекламы, фишинга и трекинга, в котором каждая функция объединена в единую экосистему через подписку.  

ClayShield сочетает 5 функций:  
- AdBlocker  
- Wi-Fi Security  
- SmartGuard AI  
- Safe Vault  
- Password Manager  

и уникальный визуальный стиль — **Clay (Soft 3D)**: “мягкие” элементы, тактильная глубина, спокойная анимация.

## 2. Основные пользовательские сценарии

| ID | Сценарий | Результат |
|----|-----------|------------|
| U1 | Включение защиты с Dashboard | Все функции активируются, если пользователь купил подписку |
| U2 | Проверка Wi-Fi безопасности | Пользователь получает статус сети (Secure / Unsecure) |
| U3 | SmartGuard-анализ сайта | Пользователь вводит URL и получает Trust Score |
| U4 | Face ID вход в Safe Vault | Пользователь получает доступ к скрытым файлам |
| U5 | Генерация и сохранение пароля | Пользователь создаёт пароль и видит уровень его надёжности |
| U6 | Просмотр и изменение настроек | Управление Face ID, уведомлениями и автоудалением данных |
| U7 | Покупка подписки | Разблокировка всех функций и отключение Paywall |

---

## 3. Бизнес-правила

1. Любое включение функции вызывает **Paywall**, если пользователь не Pro.  
2. Одна покупка → разблокировка всех функций (single unlock).  
3. 3-дневный бесплатный триал перед первой оплатой.  
4. Все данные хранятся **локально** (без серверов).  
5. После успешной покупки `isProUser = true` сохраняется локально и восстанавливается через Apphud.  
6. После Face ID входа в Vault → при сворачивании приложения сессия сбрасывается.  
7. SmartGuard Trust Score рассчитывается локально (mock до релиза API).  
8. Offline-режим: Paywall и SmartGuard недоступны, UI работает в демо-режиме.

## 4. Краткое описание UI

| Зона | Элемент | Особенности |
|------|----------|-------------|
| Главный экран | Большая кнопка-щит (Clay Button) | Tap → активирует защиту или вызывает Paywall |
| Плитки модулей | 4 блока (SmartGuard, Ad Cleaner, Privacy Shield, Safe Browsing) | Clay-карточки, выпуклые, с эффектом нажатия |
| Вкладки | 4 таба (Home, Wi-Fi, Vault, Passwords) | Clay TabBar с мягким выделением |
| Paywall | 3 экрана: Intro → Confirm → Success | Плавные переходы, выпуклые CTA |
| Safe Vault | Экран Face ID + список приватных файлов | Отдельная логика Face ID и Fake Vault |
| Settings | Clay-карточки, Learn More → Paywall | Хранение toggles в UserDefaults |
